# Release v0.3.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close GitHub issues #3, #4, #5, #6 in one release: junction admin UI, real (source-tagged) multi-org, aggregated+faceted financial table, and an org-wide entitlements tab (first LWC + Apex in the package).

**Architecture:** Four work-streams, sequenced by file collision on the wave artifacts — #5 (isolated declarative) → #4 (recipe/dataflow/dashboard restructure + `Org__c` drop) → #6 (financial table refine) → #3 (new dashboard tab: embedded LWC over `OrgLimits` + `TenantUsageEntitlement`). Metadata/dashboard tasks use deploy-and-validate cycles against a scratch org; Apex/LWC uses TDD. Multi-org (#4) has a hands-on two-org validation checkpoint the human runs with you.

**Tech Stack:** Salesforce unlocked package (`LicenseManager`), API 67.0, CRM Analytics (dataflow `.wdf` / recipe `.wdpr` / dashboard `.wdash`), Apex (`OrgLimits` system class), first-time LWC (jest), SFDMU sample data, `sf` CLI v2.

## Global Constraints

- **API version:** `sourceApiVersion` is `67.0` — every new `*-meta.xml` uses `<apiVersion>67.0</apiVersion>` where the type carries one.
- **Naming:** all custom metadata uses the `LM_` prefix.
- **Navigation (hub-and-spoke):** `LM_ProductLicense__c` is the ONLY tab; never add a tab for any other object. New objects/pages are reachable only via related lists / links.
- **pt_BR translations:** every field/object add or remove must be mirrored under `force-app/main/default/objectTranslations/…-pt_BR/`.
- **Package build + FLS:** Apex tests that set FLS-restricted fields must run inside `System.runAs(u)` where `u` holds the `LM_LicenseManager` permission set AND is built with `ProfileId = UserInfo.getProfileId()` (the package-build runner has no permission set assigned). Reference memory: `packaging-fls-test-gotcha`.
- **Version bump:** at release, set `sfdx-project.json` `versionNumber` to `0.3.0.NEXT` and `versionName` accordingly.
- **Commits:** frequent, conventional-commit style, each ending with the `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>` trailer.
- **`.forceignore`:** `profiles/**`, `package.xml`, LWC config/test files are excluded from source tracking — do not rely on them deploying.

```apex
public with sharing class LM_OrgLimitsController {
  public class LimitRow {
    @AuraEnabled
    public String name;
    @AuraEnabled
    public Integer used;
    @AuraEnabled
    public Integer max;
  }
  public class EntitlementRow {
    @AuraEnabled
    public String setting;
    @AuraEnabled
    public Decimal amountUsed;
    @AuraEnabled
    public Decimal amountAllowed;
    @AuraEnabled
    public String frequency;
  }

  @AuraEnabled(cacheable=true)
  public static List<LimitRow> getLimits() {
    List<LimitRow> rows = new List<LimitRow>();
    for (String key : OrgLimits.getMap().keySet()) {
      System.OrgLimit ol = OrgLimits.getMap().get(key);
      LimitRow r = new LimitRow();
      r.name = ol.getName();
      r.used = ol.getValue();
      r.max = ol.getLimit();
      rows.add(r);
    }
    return rows;
  }

  @AuraEnabled(cacheable=true)
  public static List<EntitlementRow> getEntitlements() {
    List<EntitlementRow> rows = new List<EntitlementRow>();
    for (TenantUsageEntitlement e : [
      SELECT MasterLabel, Setting, AmountUsed, CurrentAmountAllowed, Frequency
      FROM TenantUsageEntitlement
      WITH SECURITY_ENFORCED
    ]) {
      EntitlementRow r = new EntitlementRow();
      r.setting = e.Setting != null ? e.Setting : e.MasterLabel;
      r.amountUsed = e.AmountUsed;
      r.amountAllowed = e.CurrentAmountAllowed;
      r.frequency = e.Frequency;
      rows.add(r);
    }
    return rows;
  }
}
```

```javascript
import { LightningElement, wire } from "lwc";
import getLimits from "@salesforce/apex/LM_OrgLimitsController.getLimits";
import getEntitlements from "@salesforce/apex/LM_OrgLimitsController.getEntitlements";

export default class LmOrgLimits extends LightningElement {
  limits = [];
  entitlements = [];
  error;

  @wire(getLimits)
  wiredLimits({ data, error }) {
    if (data) {
      this.limits = data.map((r) => ({
        ...r,
        pct: r.max > 0 ? Math.round((r.used / r.max) * 100) : 0
      }));
    } else if (error) {
      this.error = error;
    }
  }

  @wire(getEntitlements)
  wiredEntitlements({ data, error }) {
    if (data) {
      this.entitlements = data;
    } else if (error) {
      this.error = error;
    }
  }
}
```
